# website-health-monitor
A simple Python script that monitors website availability by checking HTTP status and response time.   Built to practice DevOps/SRE fundamentals such as monitoring, logging, and basic alerting.

## Features
- Checks website availability
- Logs status and response time
- Prints alerts when a site is down

## Tech
Python, requests

## How to Run
```bash
pip install requests
python monitor.py
